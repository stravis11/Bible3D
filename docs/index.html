How to use it

  • Drag to orbit, scroll to zoom.
  • Click a node to center it.
  • Hover a node or link to inspect details.
  • Use book view first. Verse view is dense by design.

Notes

This uses weighted cross references from OpenBible's public dataset, primarily sourced from the Treasury of Scripture Knowledge. It is a map of connections, not a claim that every edge is a formal quotation.
